# Project Draelin

Personal AI agent workspace for the Draelin bot (draelin_bot#9763).

## Contents

- `images/` — Generated images from Nano-Banana (Google Imagen 4)
  - `what-is-a-skill.png` — Visual representation of an OpenClaw skill
  - `draelin-bot-avatar.png` — Realistic avatar for the Draelin bot
- `skills/` — Custom skill definitions for the OpenClaw agent
  - `imagen-skill.md` — Imagen 4 image generation skill

## Stack

- OpenClaw gateway (local, loopback)
- Discord + Telegram channels
- Google Gemini / Imagen 4 (Nano-Banana) for image generation
- Mission Control dashboard at `http://127.0.0.1:3000`
